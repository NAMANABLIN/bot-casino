Папблик: https://vk.com/botcasinopk
Сервер находится на сервисе ihor-hosting

В папке data, содержатся классы, функции, относящиеся к базе данных. База одна, в ней
содержится id, никнейм, деньги, использованные промокоды, пол
пользователей также в базе записывается работает ли пользователь в данный момент.

В папке defs_vk находятся обработчики команд, которые приходят боту в
сообщениях. Обработчики представляют собой асинхронные функции на которых
накинуты декораторы из класса Blueprint, этот класс используется, чтобы в main.py
можно было легко использовать эти обработчики.

В папке images есть папки lose и win, в них содержатся изображения для рулетки, но в
самом боте они не используются, они уже загружены на сервер в вк и когда нужны
отправляются с помощью их id.

В config'e  содержатся константы которые используются в других скриптах

В helper_funcs содержатся функции которые используются в другиз скриптах

Так же в папке проекта есть файл .env в нём написан токен паблика в таком формате:
VK_MAIN=сам токен

Так же в папке проекта есть база данных user.sqlite, если её нет, то при запуске бота,
она появится.
